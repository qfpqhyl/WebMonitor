# 早期脚本版本

这个目录包含 WebMonitor 项目的早期纯 Python 脚本版本，用于网页监控和邮件通知。

## 文件说明

- `main.py` - 主要的网页监控脚本
- `outlooknotice.py` - 邮件发送功能模块
- `.env` - 环境变量配置文件

## 使用说明

这些是早期的单体脚本版本，现在已经迁移到前后端架构。

如果要运行这些脚本：

1. 安装依赖：`pip install -r ../requirements.txt`
2. 配置环境变量：编辑 `.env` 文件
3. 运行主程序：`python main.py`

## 注意事项

- 这些脚本仅用于参考和备份
- 新版本将采用前后端分离架构
- 建议使用新版本进行开发
